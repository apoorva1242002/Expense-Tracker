# Expense Tracker

A full-stack Expense Tracker web application built using React, Node.js, Express, and MySQL.

## Features
- Add expenses manually
- Store expense data in MySQL
- Fetch expenses from backend API
- Import expense-like data from external API
- Visual expense breakdown using charts

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MySQL
- Charts: Recharts

## Project Structure
- `backend/` → Express server + MySQL connection
- `vite-project/` → React frontend

## How to Run

### Backend
```bash
cd backend
npm install
node server.js
